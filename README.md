# EAI-COP：让任何AI成为学生的专属学伴

> 一份“学习护照”，随身携带；任意AI，都是你的专属学伴。

## 一句话说明
EAI-COP是一套开放协议，让学生的学习档案（护照）在不同AI平台间自由流转，同时强制执行不可协商的安全基线。

## 快速开始（学生版）
1. 将本仓库中的 [系统指令书](docs/protocol.md) 全文发送给任意AI
2. 发送你的【建立学习关系】画像
3. 保存AI返回的【学习护照】——下次直接粘贴，AI瞬间恢复状态
