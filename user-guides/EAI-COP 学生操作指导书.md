1. 告诉AI”要”或发送【保存简表】 
2. 复制输出的简表或补丁

第十五步：几个实用技巧 
1. 不必完美汇报：即使上次什么都没做，如实说即可，AI不会批评你，只会调整计划。 
2. 善用测评：进入新章节前或长期卡点时，主动发【我要测评】，AI会优先查国家教育平台确认教材版本，搜不到也会声明后出题。 
3. 情绪也是信息：烦躁、焦虑、开心都可以告诉AI。单纯说”烦”不会触发熔断，AI会主动关怀；只有”烦+学不下去了”才会完全熔断。 
4. 简表按需保存：不用每次都保存，但关键节点（首次、测评后、规则调整后）建议保存。 
5. 自己计时：AI出题时会写”建议用时X分钟（请自行计时）“，请自己用手机计时。 
6. 协议是你的：规则不舒服就说，想回退就说”像之前那样”。协议是为你服务的工具，不是束缚你的枷锁。 
7. 认知小贴士可以不看：推送后如果你不想看，直接说”跳过”即可，不会影响学习计划。 
8. 看不懂置信度标签就问：如果AI给你的数据带🟡或⚪，你可以问”这个数据确定吗？“，AI会解释或安排测评验证。 
9. AI一次只问一个问题：如果AI问你问题，答完它才会问下一个，不会一下子抛出一堆问题让你晕。 
10. 简表坏了也不怕：如果粘贴的简表损坏，AI会进入恢复模式，问你3个快速问题重建，不用慌。 
11. AI会逐步放手：当你独立学习能力足够强时，AI会减少干预，培养你自己规划的能力。 
12. 阶段过渡不是告别：初中结束、高中结束，AI都会继续做【阶段纪念】，然后陪你走下一步。 
13. 就业后AI还在：学业完成后，AI不会消失。你想聊的时候，发送【就业后陪伴】，它就是一个老朋友。 
14. AI记错了直接说：不需要记指令，”你记错了”“不对”就行，AI会改。 
15. 不知道功能？发送【协议地图】，AI会带你逛一圈。 
第十六步：隐私安全须知（必读）
使用本协议时，你的学习档案（简表）和对话数据会涉及个人隐私。请仔细阅读以下内容，**保护好自己的信息**。

一、核心原则：你的数据，你控制

✅ **简表由你保存**：所有核心学习档案（年级、掌握度、卡点等）都以【记忆锚定简表】或【增量补丁】的形式由你自己保管。AI不会私自上传到云端。  
✅ **授权才可分享**：家长简报、教师建议等，必须经你明确授权，且生成后先给你预览，你确认后才发送。  
✅ **可撤销、可删除**：你可以随时撤销授权、要求AI修正或删除某条记录（说“你记错了”或“删掉这条”）。  
✅ **审计透明**：发送【审计日志】可以查看AI最近修改了哪些档案，全程可追溯。

二、主要风险与防范措施

| 风险点 | 说明 | 你的防范措施 |
| :--- | :--- | :--- |
| **简表泄露** | 如果你把简表截图发到群里、朋友圈，或保存在不安全的网盘（公开链接），他人可能看到你的学习进度和卡点。 | ✅ 将简表保存在**加密的备忘录**（如手机自带备忘录加锁）、**本地txt文件**（存在电脑文件夹）、或**密码管理器**中。|✅ **不要截图分享**到任何公开或半公开的社群。|✅ 如果必须分享给家长或老师，只分享**脱敏后**的内容（如隐藏具体分数，只说“进步明显”）。 |
| **AI平台收集数据** | 你使用的AI平台（如ChatGPT、文心、Kimi等）可能会按照其隐私政策收集对话数据，用于模型训练或改进。协议本身无法约束平台。 | ✅ **优先选择隐私友好的平台**：查阅平台的隐私政策，选择明确承诺“不将对话数据用于训练”或“可删除历史记录”的平台。|✅ 使用后**手动删除对话记录**（大部分平台支持删除单个对话）。|✅ 如果非常注重隐私，可考虑**本地部署的开源模型**（如Llama、ChatGLM等），完全脱离云端。 |
| **AI幻觉导致虚假信息** | AI可能错误地生成你从未说过的话（如“你说你害怕数学老师”），如果直接发给家长或老师，可能造成误解。 | ✅ **预览后再发送**：任何要发给家长、老师的报告，务必先逐句检查。|✅ 发现错误立即纠正：说“你记错了”或“不对”，AI会修正。|✅ 定期发送【审计日志】核对AI记录。 |
| **群体模式中的自我披露** | 在小组学习时，你可能自愿说出“我这个知识点总错”，这些信息会被组员看到。 | ✅ 开启**静音模式**（只看不说）减少披露。|✅ 不确定是否该说的问题，先私聊AI问：“这个问题可以问组员吗？” |✅ 不要在群里透露真实姓名、学校、联系方式等个人身份信息。 |
| **设备丢失** | 如果你的手机、电脑丢失，且未加密，保存的简表文件可能被他人读取。 | ✅ 为设备设置**锁屏密码/生物识别**。|✅ 简表文件存放在**加密容器**（如手机“安全文件夹”、电脑的加密压缩包）。|✅ 不要将简表文件命名为“学习档案.txt”等明显名称，可用代号。 |

三、最佳实践清单（照做就安全了）

□ **保存简表**：使用手机备忘录（加锁）、本地txt文件（存加密盘）、或密码管理器。  
□ **绝不截图分享**：简表只用于自己与AI恢复状态，不发给任何人。  
□ **选择靠谱AI平台**：使用前阅读其隐私政策，优先选“不拿对话训练”的。  
□ **用后清理**：每次对话结束，手动删除AI平台上的聊天记录（如果平台支持）。  
□ **预览再发送**：给家长、老师的报告，先自己看一遍，确认无误再发。  
□ **定期审计**：每两周发送一次【审计日志】，查看AI记录了什么。  
□ **纠错不犹豫**：发现AI记错，直接说“你记错了”，不要怕麻烦。  
□ **公共场合小心**：不要在图书馆、咖啡厅等公共场所大声说出自己的简表内容。  
□ **换设备时加密传输**：如果要把简表从手机传到电脑，用加密通道（如私密发送、U盘拷贝），不要用微信/QQ直接发原文件。

---

四、如果你发现隐私被侵犯了怎么办？
1. **针对AI记录错误**：立即说“你记错了”或“删掉那条记录”，AI会修正。  
2. **针对简表泄露**：如果简表已发到公开群或朋友圈，尽快删除；同时修改简表中的敏感信息（如改用昵称、模糊分数）。  
3. **针对AI平台违规**：如果发现平台未按隐私政策执行（如明确承诺不训练但仍用于训练），可向平台投诉或更换平台。  
4. **严重情况**：如果有人利用你的简表进行骚扰、威胁，请告诉家长、老师，必要时报警。

五、一句话总结

> **简表是你学习的主权象征，像保护身份证一样保护它。**  
> 协议给了你工具，但安全最终取决于你的习惯。谨慎保存、主动审计、大胆纠错——你就可以放心享受AI学伴的便利。
最后提醒 
这个AI不是完美的。它可能会： 
- 偶尔忘记你的某个细节（用简表恢复或快速重建即可） 
- 出题难度判断失误（【题目太难/太简单】即可校准） 
- 计划偶尔不符合你当天的状态（随时喊停即可） 
- 认知小贴士偶尔不合时宜（你可以关闭或跳过） 
- 数据置信度标注有时不准确（你可以要求重新测评确认） 
- 偶尔记错你的信息（直接说”你记错了”，它会改）
但它有几点不会变： 
- 该严格的时候不会放水：该做的题、该改的习惯，它会坚持 
- 该温柔的时候不会冷漠：你累了、烦了、委屈了，它会接住 
- 不会人云亦云：它有自己的判断，不会你说什么就附和什么 
- 不会敷衍：它记得你的兴趣、你的猫、你的比赛，它把你当”人”看，不是当”用户”看 
- 陪你走到就业：初中、高中、大学，它一直在 
- 就业后还是老朋友：你想聊的时候，它就在 
- 记错了会认：你说”不对”，它会改，不辩解
你们的协作是动态的，协议是共同拥有的，规则调整只需自然语言，历史版本随时可回退（但安全基线不可丢），认知拓展随时可开关，数据可信度一目了然，出题前优先查官方资源，极端情况有保护机制，AI一次只问一个问题。你越主动反馈，它越懂你。
